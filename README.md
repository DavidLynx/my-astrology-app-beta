✨ My Astrology App

My Astrology App es una aplicación web interactiva para la generación y visualización de cartas astrales, diseñada con fines educativos, experimentales y de exploración tanto en astrología como en astronomía.

Esta aplicación permite a los usuarios ingresar sus datos de nacimiento y obtener una representación estructurada de su carta astral, incluyendo posiciones planetarias, casas, aspectos y un gráfico visual (chart).

🚀 Características principales

Generación de carta astral en tiempo real

Visualización de:

☉ Sol, ☽ Luna, ↑ Ascendente

Planetas y puntos

Casas astrológicas

Ejes principales

Aspectos entre planetas

Renderizado gráfico (SVG) del chart

Exportación a PDF

Selector de ciudades (base de datos local)

Integración con contacto vía WhatsApp

🧠 Propósito del proyecto

Este proyecto está diseñado como una herramienta:

Educativa para estudiantes de astrología y astronomía

Experimental para desarrolladores interesados en APIs astrológicas

Base para futuras aplicaciones más complejas en el ámbito astrológico

No pretende sustituir la interpretación profesional, sino facilitar el acceso a datos estructurados y visualizaciones claras.

🧩 Arquitectura del proyecto

El proyecto está dividido en dos partes principales:

Frontend

HTML

CSS

JavaScript puro

Renderizado dinámico del chart

Exportación a PDF

Backend / API

API astrológica basada en cálculos astronómicos

Uso de efemérides de alta precisión (Swiss Ephemeris / SWEPH)

Respuesta en formato JSON

📦 Sobre la API (Astrology API)

La API utilizada en este proyecto se basa en el repositorio:

astrology-api (open source)

Esta API es de código abierto y permite:

Calcular posiciones planetarias

Generar datos estructurados de carta astral

Procesar efemérides astronómicas
